FAQ

FAQ
Does this portal bypass Chaoxing login?

No. It only aggregates entry points. It does not proxy requests, automate login, or collect account credentials.

Why do some links open a login page first?

If your browser is not already signed in to the required Chaoxing service, the target site will redirect you to its own login flow before showing the destination page.

Can I install this portal like an app?

Yes. The home page supports PWA installation. Chromium browsers usually surface a native install entry, while Safari uses manual flows such as Add to Home Screen or Add to Dock.
